Em grupos os alunos devem considerar 2 cenários : 
(i) sistema de registro de notas de alunos para
uma instituição de ensino pequena com algumas centenas de alunos e 
(ii) sistema de registro de notas
de alunos para uma instituição de ensino com centenas de milhares de alunos e unidades espalhadas
por todo o país. Para cada um destes cenários, o grupo deve elencar as vantagens e desvantagens do
uso de RAD para o cenário, e decidir se eles recomendam o uso de RAD para o cenário
