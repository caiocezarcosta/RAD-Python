Questão 1) A linguagem de programação Python tem enorme facilidade e flexibilidade para a
realização de consultas nos mais variados banco de dados. Qual comando abaixo é usado para
rodar uma consulta:
a) execute()
b) rodar()
c) executar()
d) run()
e) show()

Resposta: Letra (A)
